# Teaching Guides

This directory contains reproducible, bilingual teaching guides for installing and running Jupyter Notebook on different platforms.  
Each guide provides step-by-step instructions, prerequisites, and notes for classroom use.

## Available Guides

- [Install Jupyter Notebook on Ubuntu 24.04](install-jupyter-notebook/install-jupyter-ubuntu24/README.md)  
  Covers installation using **pip** (virtual environment) and **APT** package manager.

- [Install Jupyter Notebook on Windows](install-jupyter-notebook/install-jupyter-windows/README.md)  
  Covers installation using **Anaconda** and **Microsoft Store**.

---

## Notes
- Guides are designed for clarity and reproducibility.  
- Each folder includes a short overview (`README.md`) and a detailed step-by-step guide (`Install_Jupyter_*.md`).  
- Documentation is bilingual (English + Persian) to support wider accessibility.
