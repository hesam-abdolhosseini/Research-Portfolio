# Install Jupyter Notebook on Ubuntu 24.04

This teaching guide explains how to install and run Jupyter Notebook on Ubuntu 24.04 using two methods: **virtual environment** (recommended) and **system‑wide installation**.  
It is designed for reproducibility and classroom use.
