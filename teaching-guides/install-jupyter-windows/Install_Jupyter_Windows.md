# Install Jupyter Notebook on Windows

## Prerequisites
- Windows 10 or 11  
- Python 3.7 or higher installed  

---

## Method 1: Install via Anaconda (recommended)

1. **Download Anaconda**  
   Go to [Anaconda’s official website](https://www.anaconda.com/) and click the **Free Download** button at the top of the page.  
   *(The installer is ~900 MB.)*

2. **Run the installer**  
   Follow the prompts, agree to the terms, and choose the installation location.  
   ⚠️ Do **not** add Anaconda to your PATH environment variable.

3. **Launch Jupyter Notebook**  
   - Open the **Start menu** and search for **Anaconda Navigator**.  
   - Launch it, then click the **Jupyter Notebook** button to install and open the notebook interface.

---

## Method 2: Install via Microsoft Store

1. **Open Microsoft Store**  
   Search for **Jupyter Notebook**.

2. **Install**  
   Click the **Install** button to download and set up Jupyter Notebook.

3. **Launch**  
   After installation, open the **Start menu**, search for **Jupyter Notebook**, and launch it.

---

## Notes
- The **Anaconda method** is recommended because it provides a clean environment with many scientific libraries preinstalled.  
- The **Microsoft Store method** is simpler but may not include all dependencies.  
- Always verify your Python installation with:  
  ```bash
  python --version
  ```
