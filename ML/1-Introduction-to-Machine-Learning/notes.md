# Introduction to machine learning — bilingual notes

## What is machine learning?
- **Definition:** Algorithms that learn patterns from data to make predictions or decisions without being explicitly programmed for each task.
- **Examples:** Healthcare (cancer detection), banking (fraud detection), telecom (churn prediction), recommendation engines.

> تعریف: الگوریتم‌هایی که از داده‌ها الگو می‌آموزند تا پیش‌بینی یا تصمیم‌گیری کنند، بدون برنامه‌نویسی صریح برای هر حالت.
> مثال‌ها: سلامت (تشخیص سرطان)، بانکداری (کشف تقلب)، مخابرات (ترک مشتری)، سیستم‌های توصیه‌گر.

## Supervised vs. unsupervised
- **Supervised:** Labeled data; tasks include regression and classification.
- **Unsupervised:** Unlabeled data; tasks include clustering and dimensionality reduction.

> نظارت‌شده: داده‌های برچسب‌دار؛ وظایف شامل رگرسیون و طبقه‌بندی.
> بدون‌نظارت: داده‌های بدون برچسب؛ وظایف شامل خوشه‌بندی و کاهش ابعاد.

## Core Python libraries
- **NumPy/Pandas:** Data handling
- **scikit-learn:** Models and evaluation
- **Matplotlib/Seaborn:** Visualization
- **SciPy:** Scientific utilities

> کتابخانه‌ها: NumPy/Pandas (داده)، scikit-learn (مدل‌ها)، Matplotlib/Seaborn (تصویرسازی)، SciPy (ابزار علمی).
