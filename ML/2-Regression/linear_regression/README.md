# Simple Linear Regression Lab

This folder contains a reproducible lab on **Simple Linear Regression** using scikit‑learn, aligned with the IBM ML0101EN course.

## Contents
- `FuelConsumptionCo2.csv` → dataset of vehicle fuel consumption and CO₂ emissions  
- `LAB_Simple_Linear_Regression.ipynb` → notebook with lab instructions  
- `LAB_Simple_Linear_Regression.pdf` → static version of the lab  
- `LAB_Simple_Linear_Regression_with_output.ipynb` → notebook with executed outputs  
- `LAB_Simple_Linear_Regression_with_output.pdf` → static version with outputs

## How to Run

1. Make sure Jupyter Notebook is installed on your system.  
   - On Linux/macOS: run `jupyter notebook` from the terminal.  
   - On Windows: run `jupyter notebook` from Anaconda Prompt or Command Prompt.

2. Download the `.ipynb` lab file (e.g., `LAB_Simple_Linear_Regression.ipynb`) and place it inside your Jupyter Notebook working directory.

3. Launch Jupyter Notebook:
 ```bash
   jupyter notebook
```
This will open the Jupyter interface in your browser.
- Open the lab notebook file from the Jupyter interface.
- Run each cell step‑by‑step.
- The notebook includes commands to install required dependencies (numpy, pandas, matplotlib, scikit‑learn).
- Outputs will be generated as you execute each section.

## Notes
- No manual installation of packages is required — the notebook handles dependencies.
- Works the same way on Linux and Windows.
- If you want a clean environment, you can still create a virtual environment, but it’s optional.
