# Regression

This module covers regression techniques in machine learning, aligned with the IBM ML0101EN course.  
It includes reproducible labs, datasets, and bilingual documentation for clarity and accessibility.

## Contents
- **linear_regression/** → Lab on Simple Linear Regression using scikit‑learn  
  - `LAB_Simple_Linear_Regression.pdf` → step‑by‑step lab instructions  
  - `FuelConsumptionCo2.csv` → dataset used in the lab

Future subfolders will include:
- Multiple Linear Regression
- Non‑linear Regression
- Polynomial Regression
- Additional regression methods as needed

## Notes
- Each subfolder contains a minimal README, datasets (if any), and concise notebooks/scripts.  
- Labs are designed for reproducibility and classroom use.  
- Documentation is bilingual (English + Persian).
